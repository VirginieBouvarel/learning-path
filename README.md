# KataSensei — Learning Path

Repo du site statique de formation pour construire KataSensei.

## Structure

- `public/` : site public HTML/CSS/JS
- `content/` : export Notion et sources éditoriales conservées
- `docs/` : specs et notes internes de construction
- `AGENTS.md` : règles de travail pour l'agent

## Entrée locale

Ouvrir le site via :

- `public/index.html`

ou servir `public/` avec Live Server.
