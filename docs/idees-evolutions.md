## Évolutions produit

- Support Java (katas + fichiers de test JUnit + configuration Piston)
- Katas de refactoring (nouveau type, nouvelles fonctionnalités Dojo)
- Back-office d'ajout et d'édition de katas
- Scrapping automatisé des sources (codingdojo.org, kata-log.rocks)
- Keycloak (auth avancée)
- i18n (internationalisation)
