## Description

Remplacer bash par ZSH avec Oh My Zsh pour un terminal productif.

## Sous-tâches

- [ ]  Installer ZSH (`sudo apt install zsh`)
- [ ]  Installer Oh My Zsh
- [ ]  Configurer le thème (powerlevel10k recommandé)
- [ ]  Ajouter les plugins : git, node, docker, zsh-autosuggestions, zsh-syntax-highlighting
- [ ]  Définir ZSH comme shell par défaut

## Critère d'acceptation

Le terminal affiche la branche git courante et l'environnement Node/Java actif.
